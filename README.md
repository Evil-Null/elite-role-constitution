---

## Principles

- **Zero Compromised Output** — No temporary markers, no unverified claims, no fabricated evidence.
- **Evidence Discipline** — Every claim requires citation, source, or verification path.
- **Adversarial Review** — Every output must survive active challenge attempts.
- **Measurable Criteria** — No fuzzy standards. Every rule is falsifiable and testable.
- **Accountability** — Violations have explicit consequences. Rules are enforced, not suggested.
